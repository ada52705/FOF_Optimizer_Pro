# FOF Optimizer Pro

自有资金 FOF 组合研究网页原型，用于展示中国市场多策略资产池的组合构建、动态调仓、风险贡献、相关性压力和区间业绩分析。

## 项目用途

本项目面向金融机构自有资金投资研究场景，支持以公募基金、主观私募、量化私募和策略指数为底层资产，构建目标收益约束下的 FOF 组合原型。

当前版本为纯静态网页，无需后端服务。

## 当前功能

- 选择全资产池或核心资产池。
- 设置目标收益、调仓频率、回看窗口、风险口径和组合模型；默认参数为月度调仓、2 年回看窗口、协方差风险口径。
- 支持历史回看均值方差、风险平价和 Black-Litterman 组合模拟。
- 支持输入投资经理对各类资产的年化收益和最大回撤预期。
- 组合净值图支持年度选择和自定义区间，组合和沪深300在所选区间重新归一到 1。
- 自动计算所选区间收益、区间年化收益、最大回撤、年化波动和区间夏普。
- 动态权重占比图和风险贡献面积图会随所选区间同步裁剪。
- 区间月度收益按年度分行，最后一列展示“年度”收益。
- 资产风险收益画像包含逐年收益小柱图，正收益向右红色，负收益向左绿色，并展示日度 VaR (95%)。
- 正常相关性和极端相关性热力图位于页面末尾。

## 组合计算流程

本原型采用“滚动回看 + 定期调仓”的组合回测方式。回看窗口不是锁定最初一段历史，也不是前 1 年或前 2 年权重不变，而是在每个调仓日重新向前滚动取样。

1. 每个交易日先判断是否到达调仓日。默认调仓频率为月度，也可以切换为季度。
2. 如果到达调仓日，系统取该调仓日前最近 N 个交易日作为估计样本。半年约 126 个交易日，一年约 252 个交易日，两年约 504 个交易日。
3. 用该滚动窗口估计各资产预期收益、波动率、相关性和协方差矩阵。风险口径默认为协方差，也可以切换为半方差。
4. 根据所选组合模型和约束条件求出新权重。约束包括单资产上限、债券底仓下限、权益方向上限和目标收益。
5. 新权重从调仓日开始持有到下一次调仓日。区间内每日组合收益 = 当期权重 × 当日各资产收益。
6. 用每日组合收益累乘得到组合净值，并计算回撤、年化收益、年化波动、夏普、风险贡献和月度收益。

默认参数建议为：月度调仓、两年回看窗口、协方差风险口径。月度调仓更接近 FOF 组合的申赎和管理可行性；两年窗口比一年更稳，能降低单一年份行情对均值和相关性的过拟合。一年窗口适合作为敏感性测试，半年窗口更适合观察市场结构快速变化时的战术版本。

## 三种组合模型

### 均值-方差优化

默认参数：目标收益为滑杆设定值，默认 8%；回看窗口默认两年；风险口径默认协方差；单资产上限、债券底仓下限和权益上限使用左侧约束输入。

流程：在每个调仓日，用滚动窗口估计各资产年化收益和协方差矩阵；随机生成大量候选权重；剔除不满足约束的组合；对剩余组合计算预期收益、组合波动率和目标收益偏离；选择综合得分最低的权重。得分核心是降低波动、贴近目标收益，并惩罚低于目标收益的组合。

### Black-Litterman

默认参数：历史收益作为先验收益；投资观点输入为空时，不主动覆盖历史先验；如果输入某类资产的预期收益，则当前原型按“40% 历史先验 + 60% 投资观点”混合为该资产的新预期收益；预期波动率输入用于风险惩罚项；回看窗口和风险口径仍然生效。

流程：在每个调仓日，先用滚动窗口估计历史先验收益和协方差矩阵；再读取“投资观点输入”中的预期收益和预期波动率；有观点的资产会用观点收益混合历史收益，没有观点的资产保留历史先验；随后进入与均值-方差类似的候选权重搜索，在目标收益、波动、约束和观点风险惩罚之间选择权重。

注意：如果没有输入任何观点，理论上 Black-Litterman 在当前原型中应接近均值-方差优化，因为收益先验没有被观点调整。如果页面上仍看到小幅变化，主要原因是当前优化器使用随机候选权重搜索，每次刷新候选组合不同；后续可以加入固定随机种子或确定性优化器，使同一参数下结果完全可复现。

### 风险平价

默认参数：风险贡献目标为等比例分布，即每个入选资产承担接近相同的总风险贡献；回看窗口默认两年；风险口径默认协方差，也可以切换半方差；仍然遵守单资产上限、债券底仓下限和权益上限。

流程：在每个调仓日，用滚动窗口估计协方差或半方差矩阵；随机生成候选权重；剔除不满足约束的组合；计算每个资产的总风险贡献 TRC；选择风险贡献最接近等权目标、同时兼顾组合波动和观点风险惩罚的权重。风险平价不以“权重相等”为目标，而是让风险贡献尽量均衡。

### 风险口径说明

协方差口径使用全部日收益估计风险矩阵，既惩罚上涨波动也惩罚下跌波动。半方差口径只使用下跌收益构造风险矩阵，更强调回撤和亏损方向的风险。
